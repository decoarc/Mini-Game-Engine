# Mini-Game-Engine

A study on how to make a game engine.

## Compilation

To compile the project, use the following command:

```bash
g++ -o gameEngine.exe src/main.cpp src/game.cpp src/window.cpp src/input.cpp src/render.cpp -lgdi32 -luser32 -lkernel32
```

## Execution

To run the Game Engine:

```bash
.\gameEngine.exe
```
